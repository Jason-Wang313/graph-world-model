# Claims

## Supported

- Exact finite tie-aware Best-of-N laws predict selected utility for finite graph-physics candidate pools.
- Controlled graph-physics high-`N` selection can expose selected-tail failures: internal score improves while real utility remains below the oracle tail.
- Repair components have measurable roles: observed-energy penalties, action penalties, calibration-gap penalties, and constraint probes each address different risk surfaces.
- Combined repair and adaptive gating improve high-`N` selection in the synthetic stress suite.
- A CPU NumPy learned-lite calibrator improves held-out rank alignment between candidate scores and real utility.
- The stress suite covers multiple graph families, hidden failures, and stress levels inside a CPU-local synthetic benchmark.

## Unsupported

- Real-robot validation.
- External physics-engine benchmark superiority.
- State-of-the-art performance.
- Universal improvement from increasing `N`.
- Deployment safety claims beyond the controlled synthetic setting.
